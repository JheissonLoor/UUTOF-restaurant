# Alcance de planes UTTOF v3

Este documento evita mezclar los nombres históricos Básico/Premium de algunos handoffs con el modelo comercial vigente.

| Capacidad | Estándar | Operación Pro | Multi-local |
| --- | --- | --- | --- |
| Experiencia | Una web para cuatro perfiles | Cuatro aplicaciones dedicadas | Aplicaciones avanzadas y varias sedes |
| Cocina | Kanban simple integrado | Kanban dedicado, origen, progreso, urgencia fija y polling 15 s | KDS oscuro, WebSocket, sonido, cronómetros e ítems individuales |
| Mesero | Verificación de efectivo | Mesas, pedidos, entrega, cuenta QR y cobro | Ranking, upselling, offline y herramientas avanzadas |
| Cliente | Carta, pedido, seguimiento y postpago | Aplicación móvil dedicada con walk-in | Pago dividido, fidelización y pedidos programados |
| Admin | Operación base | Dashboard y reportes completos en pantalla | Inventario, exportación, IA y múltiples locales |

Los archivos `PREMIUM.md` recibidos en handoffs antiguos describen capacidades que, según Planes v3, se clasifican como **Multi-local**. No deben habilitarse dentro de Operación Pro.
